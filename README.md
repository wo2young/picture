# picture

가족 구성원들이 사진과 추억을 공유할 수 있는  
**가족 사진관 개인 프로젝트**입니다.

## 프로젝트 목적
- 가족 간 사진 및 일기 공유
- 간단하고 직관적인 UI 제공
- 가족 중심의 추억 기록 앱

## 사용 기술
### Backend
- Python (FastAPI)
- PostgreSQL
- Azure Blob Storage

### Frontend
- Flutter

## 주요 기능
- 사진 업로드 및 조회
- 앨범 관리
- 일기(다이어리) 작성
- 가족 구성원 관리

## 프로젝트 특징
- 개인 프로젝트
- 기획부터 설계, 구현까지 단독 진행
